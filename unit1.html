Hardware and Software

A Computer is made up of two aspects: hardware and software. Hardware is all the physical, tangible components of the computer (e.g. mouse & keyboard, CPU, monitor). By contrast, software is a set of instructions that runs on the hardware and tells the computer what to do (e.g. Google Chrome, Windows 10, Adobe Photoshop). Hardware cannot perform any task without software while software cannot be executed without hardware. Hardware is manafactured in factories while software is created by computer programmers through the use of a programming language.
