C#

C# is a objected-oriented and component-oriented programming language created by Microsoft on the .NET framework in 2000. C# has been widely used throughout the programming industry and is modern, efficient, structured and easy to learn.

Examples of C# commands:

  • Console.ReadLine(""): Outputs the text entered within the quoted parameters

  • Console.ReadLine(""): Reads input entered within the quoted parameters

  • int x = y: Creates integer type variable x and stores value y within it.

  • for(x;y;z) {abc}: Creates a for loop where x is the counter, y is the condition and z is the iteration; the statement abc within the curly braces is executed.

  • if (x) {abc}: Creates a if statement where x is the the condition and the statement abc is executed.
